JavaScript methods

When writing your JS code there is a great chance that they will come in handy. I will keep here all the methods that I used whilst doing exercises on Code Wars.

String split() Method

The split() method is used to split a string into an array of substrings, and returns the new array.

ABCDEFG

<p id="split">ABCDEFG</p>
<button onclick="splitFunction()">Split</button>
<script>
    function splitFunction() {
        var str = 'ABCDEFG';
        var res = str.split("");
        document.getElementById("split").innerHTML = res;
    }
</script>

Array slice() Method

This method returns the selected elements in an array, as a new array object. The slice() method selects the elements starting ar the given start argument, and ends at, but does not include, the given end argument.

var fruits = ['Apple', 'Bannana', 'Lemon', 'Mango', 'Orange']

var new_fruit = fruits.slice(1,3)

<p id="slice">var fruits = ['Apple', 'Bannana', 'Lemon', 'Mango', 'Orange']</p>
<button onclick="sliceFunction()">Slice</button>
<script>
    function sliceFunction() {
        var fruits = ['Apple', 'Banana', 'Lemon', 'Mango', 'Orange'];
        var new_fruits = fruits.slice(1,3);
        document.getElementById("slice").innerHTML = new_fruits;
    }
</script>

Array reverse() Method

This method reverses the order of the elements in an array. It will change an original array.

var transport = ['Car', 'Bike', 'Plane', 'Train'];

<p id="reverse">var transport = ['Car', 'Bike', 'Plane', 'Train']</p>
<button onclick="reverseFunction()">Reverse</button>
<script>
    function reverseFunction() {
        var transport = ['Car', 'Bike', 'Plane', 'Train'];
        var rev_trans = transport.reverse();
        document.getElementById("reverse").innerHTML = rev_trans; 
    }
</script>

Arry join() Method

This method returns the array as a string.The elements will be separated by a sepcified separator. The deafualt separator is comma (,). This method will not change the original array.

var languages = ['Python', 'Java', 'C#', 'Ruby', 'PHP']

<p id="join">var languages = ['Python', 'Java', 'C#', 'Ruby', 'PHP']</p>
<button onclick="joinFunction()">Join</button>
<script>
    function joinFunction() {
        var languages = ['Python', 'Java', 'C#', 'Ruby', 'PHP'];
        var join_lang = languages.join();
    document.getElementById("join").innerHTML = join_lang;
    }

Number toString() Method

This method converts a number to a string.

var num = 13;

<p id="toString">var num = 13;</p>
<button onclick="toStringg()">toString</button>
<script>
    function toStringg() {
        var number = 13;

parseInt() Function

This function parses a string and returns an integer.Remamber, only the first number in the string is returned!

var a = '623'

<p id="parseInt">var a = '623'</p>
<button onclick="parseIt()">ParseInt</button>
<script>
    function parseIt() {
        var string = '623';
        var strToInt = parseInt(string);
        document.getElementById("parseInt").innerHTML = strToInt;
    }
</script>

Array push() Method

This method is used in order to add an element to the end of an array.

Banana,Orange,Apple,Mango

<p id="push">Banana,Orange,Apple,Mango</p>
<button onclick="pushFunction()">Push</button>
<script>
    function pushFunction() {
        var fruits = ['Banana', 'Orange', 'Apple', 'Mango'];
        document.getElementById("push").innerHTML = fruits;
        fruits.push("Kiwi");
        document.getElementById("push").innerHTML = fruits;
    }
</script>

Array unshift() Method

Banana,Orange,Apple,Mango

<p id="unshift">Banana,Orange,Apple,Mango</p>
<button onclick="unShiftFunction()">Unshift</button>

function unShiftFunction() {
    var fruits = ['Banana', 'Orange', 'Apple', 'Mango'];

    fruits.unshift('Kiwi', 'Durian');
    document.getElementById("unshift").innerHTML = fruits;
</script>
