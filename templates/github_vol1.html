How to push newly created project to your Github account?

1) Decide where you want to keep your files and get there.

2) Check if git is already initialized: 

 git status

If it is not being tracked by git, you must initialize it: 

 git init

3) Create new repository on your Githb account. If you already have prepared a repository, skip this point.

4) Connect your project with your empty Github repository:

Copy the link of your repo on Github:

Go back to your project in the terminal and type: 

 git remote add origin [copied link of your Github repository]

Push your branch to Github 

 git push origin master

Voila! You should see your project files in Github repository!

Adding an existing project to Github

1) Get to the folder where you keep your project

2) Check if git is already initialized: 

 git status

If it is not being tracked by git, you must initialize it: 

 git init

3) Add the files to your new local repository. This stages them for the first commit: 

git add [name_file] OR git add . (to add all files)

4) Commit the files that you've staged in your local repository 

 git commit -m "your comment"

5) Copy the link of your repository on Github

6) In terminal, add the URL for the remote repository where your local repository will be pushed: 

 git remote add origin [link of your Github repository]
 git remote -v

Push the changes in your local repository to GitHub: git push origin master
