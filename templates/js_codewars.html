Convert number to reversed array of digits

Given a random non-negative number, you have to return the digits of this number within an array in reverse order.

Example: 348597 => [7,9,5,8,4,3]

Number : 

function digitize(n) {
var n = n.toString();
n = Array.from(n);
n = n.map(el=>parseInt(el));
n = n.reverse();
return n;
};

Vowel Count

Return the number of vowels in the given string. We will consider only: a, e, i, o, u.

I decided to put this exercise here as in order to solve it, you can use a loop within a loop, which is handy when comparing.

String: 

function getCount(str) {
var vowelsCount = 0;
var vowels = 'aeiou';
vowels = vowels.split('');
str = str.split('');
for (i=0;i<str.length;i++){
  for (j=0;j<vowels.length;j++){
    if (str[i] === vowels[j]){
      vowelsCount += 1;
    }
  }
};
return vowelsCount
};

Disemvowel Trolls

Your task is to write a function that takes a string and return a new string with all vowels removed.

Here we will use the replace() method.

String: 

function disemvowel(str) {
    return str.replace(/[aeiou]/gi,"");
}

Shortes Word

Given a string of words, return the length of the shortest wrod(s).

String: 

function findShort(s){
s = s.split(' ').sort()
var len = []
for (i=0;i<s.length;i++){
  len[i]=s[i].length
}
len = len.sort(function(a,b){return a-b})
return Math.min.apply(0,len)
}

Two to One

Take 2 strings s1 and s2 including only letters from ato z. Return a new sorted string, the longest possible, containing distinct letters, each taken only once - coming from s1 or s2.

This exercise shows you how to get rid of duplicates.

String 1: String 2: 

function longest(s1, s2) {
    s3 = s1.concat(s2)
    s3 = new Set(s3)
    return Array.from(s3).sort().join('')
}
