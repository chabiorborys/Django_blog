Convert number to reversed array of digits

Given a random non-negative number, you have to return the digits of this number within an array in reverse order.

Example: 348597 => [7,9,5,8,4,3]

Number : 

function digitize(n) {
var n = n.toString();
n = Array.from(n);
n = n.map(el=>parseInt(el));
n = n.reverse();
return n;
};

Vowel Count

Return the number of vowels in the given string. We will consider only: a, e, i, o, u.

I decided to put this exercise here as in order to solve it, you can use a loop within a loop, which is handy when comparing.

String: 

function getCount(str) {
var vowelsCount = 0;
var vowels = 'aeiou';
vowels = vowels.split('');
str = str.split('');
for (i=0;i<str.length;i++){
  for (j=0;j<vowels.length;j++){
    if (str[i] === vowels[j]){
      vowelsCount += 1;
    }
  }
};
return vowelsCount
};
